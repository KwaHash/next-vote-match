# わたしの政治

**政治は、わたしの"推し"から始まる**

インタラクティブな診断クイズを通じて、日本の市民が自分の価値観に合った政治家や候補者を発見できる、モダンな政治マッチングプラットフォームです。

## 🎯 概要

わたしの政治は、Next.jsベースのWebアプリケーションで、市民と政治家の間のギャップを埋めます。価値観に基づいた質問に答えることで、ユーザーは自分の信念や優先事項を共有する政治家を発見でき、政治参加をより個人的で意味のあるものにします。

## ✨ 機能

- **🎯 価値観マッチング**: ユーザーと相性の良い政治家をマッチングするインタラクティブな診断クイズ
- **👥 政治家ディレクトリ**: 政治家や候補者のプロフィールを閲覧・探索
- **📊 候補者情報**: 写真、政党所属、詳細情報を含む包括的な候補者プロフィール
- **💝 サポートシステム**: 政治家を支援・フォローするためのコミュニティ機能
- **🎨 モダンなUI**: shadcn/uiとTailwind CSSで構築された美しくレスポンシブなインターフェース
- **🌐 日本語対応**: 日本のユーザー向けに完全にローカライズ

## 📋 前提条件

- **Node.js**: バージョン18.x以上
- **npm**: バージョン9.x以上（Node.jsに含まれています）

Node.jsのバージョン管理には[nvm](https://github.com/nvm-sh/nvm)の使用を推奨します：

```bash
# nvmをインストール（まだインストールしていない場合）
# 参照: https://github.com/nvm-sh/nvm#installing-and-updating
```

## 🚀 はじめに

### 1. リポジトリのクローン

```bash
git clone <repository-url>
cd next-vote-match
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 環境変数の設定

ルートディレクトリに `.env.local` ファイルを作成：

```env
# データベース設定
DB_HOST=your_database_host
DB_USER=your_database_user
DB_PASSWORD=your_database_password
DB_NAME=your_database_name
# 必要に応じて他の環境変数を追加
```

### 4. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

## 🎨 UIコンポーネント

このプロジェクトは[shadcn/ui](https://ui.shadcn.com/)を使用しています。これはRadix UIとTailwind CSSで構築された再利用可能なコンポーネントのコレクションです。コンポーネントは`components/ui/`に配置されており、必要に応じてカスタマイズできます。

## 🔧 設定

### Next.js設定

プロジェクトには、クライアントサイドビルド用のカスタムwebpack設定と画像最適化設定が含まれています。詳細は`next.config.mjs`を参照してください。

### Tailwind CSS

Tailwindは`tailwind.config.ts`でカスタムテーマ拡張とアニメーションを使用して設定されています。

### ESLint

ESLint設定は、Next.jsの推奨設定に加えて、未使用インポート用の追加プラグインを含んでいます。

## 🗄️ データベース

アプリケーションはデータベース接続にMySQL2を使用しています。データベース設定と接続ロジックは`lib/db.ts`にあります。
